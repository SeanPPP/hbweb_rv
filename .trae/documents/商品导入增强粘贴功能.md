# 国内商品导入 - 增强粘贴功能：支持从列头粘贴

## 需求

当前粘贴必须先点击表格数据行内的输入框，用户希望**点击列头后粘贴 Excel 数据**，自动从第一行开始填充。

## 当前代码逻辑

**文件**: `src/pages/DomesticPurchase/ProductImport/index.tsx`

粘贴入口：`handlePaste`（第335-380行），全局监听 `paste` 事件。

关键逻辑：
- 第348行 `target.closest('td')`：必须找到数据行的 `<td>`，列头是 `<th>` 所以不支持
- 第352-354行：通过 `<td>` 在 `<tr>` 中的索引计算 `startColIndex = tdIndex - 1`
- 第355-356行：通过 `data-row-key` 找到行索引

## 改造方案

### 修改 `handlePaste` 函数

当用户点击了列头（`<th>`）时，支持从第一行开始粘贴：

1. 如果 `target.closest('td')` 找到 → 走现有逻辑（从点击行粘贴）
2. 如果 `target.closest('td')` 为 null，尝试 `target.closest('th')` → 从第一行开始粘贴，起始列为点击的列头对应的列

### 具体步骤

1. 在 `handlePaste` 中增加对 `<th>` 的处理分支
2. 当找到 `<th>` 时：
   - 从 `<th>` 在 `<thead>` 中的索引算出 `startColIndex`（同样减1跳过 rowSelection 列）
   - `rowIndex` 固定为 0（从第一行开始）
3. 如果 `<td>` 和 `<th>` 都找不到，才提示"请先选中一个单元格"

### 代码改动

```typescript
// 原代码第348-357行
const td = target.closest('td')
if (!td) { message.warning('请先选中一个单元格作为粘贴起点'); return }
const tr = td.closest('tr')
if (!tr) return
const allTds = Array.from(tr.querySelectorAll('td'))
const tdIndex = allTds.indexOf(td)
const startColIndex = tdIndex - 1
const startRowIndex = parseInt(tr.getAttribute('data-row-key') || '0')
const rowIndex = state.products.findIndex((p) => p.id === String(startRowIndex))
if (rowIndex === -1) { message.warning('无法确定行位置'); return }

// 改为：
const td = target.closest('td')
const th = target.closest('th')
let startColIndex: number
let rowIndex: number

if (td) {
  const tr = td.closest('tr')
  if (!tr) return
  const allTds = Array.from(tr.querySelectorAll('td'))
  const tdIndex = allTds.indexOf(td)
  startColIndex = tdIndex - 1
  const startRowIndex = parseInt(tr.getAttribute('data-row-key') || '0')
  rowIndex = state.products.findIndex((p) => p.id === String(startRowIndex))
  if (rowIndex === -1) { message.warning('无法确定行位置'); return }
} else if (th) {
  const thead = th.closest('tr')
  if (!thead) return
  const allThs = Array.from(thead.querySelectorAll('th'))
  const thIndex = allThs.indexOf(th)
  startColIndex = thIndex - 1
  rowIndex = 0
} else {
  message.warning('请先选中一个单元格或列头作为粘贴起点')
  return
}
```
